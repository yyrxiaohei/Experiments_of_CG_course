# 1. 项目说明

本版本严格以教师给出的质点弹簧模型参考代码为基础，仅对代码进行模块化拆分、命名整理和注释补充，不再加入球体碰撞、剪切弹簧、弯曲弹簧等额外功能。

# 2. 项目架构

```text
CG_Lab/
├── pyproject.toml
└── src/
    └── Work7/
        ├── __init__.py
        ├── config.py
        ├── physics.py
        ├── renderer.py
        ├── main.py
        ├── README.md
        └── images/
            └── .gitkeep
```

模块说明：

- `config.py`：集中管理窗口、布料、物理模拟、积分器和相机参数；
- `physics.py`：定义 Taichi 数据场、初始化 Kernel、力学计算函数和三种积分 Kernel；
- `renderer.py`：负责 Taichi GGUI 窗口、相机和三维场景渲染；
- `main.py`：负责程序入口、GUI 控制面板、积分方法切换和逐帧调度。

# 3. 运行方法

在项目根目录执行：

```bash
uv run -m src.Work7.main
```

# 4. 实现内容

- 构建 `20 × 20` 的布料质点网格；
- 使用水平和竖直结构弹簧连接相邻质点；
- 固定布料第一排的两个角点；
- 计算重力、速度阻尼和胡克弹簧力；
- 使用 `ti.atomic_add` 解决弹簧力并行累加冲突；
- 使用 `clamp_velocity()` 限制最大速度；
- 实现显式欧拉、半隐式欧拉和固定点迭代隐式欧拉；
- 将初始化拆分为位置、弹簧和渲染索引三个 Kernel；
- 使用 Taichi GGUI 实现三维场景、相机控制和交互面板；
- 切换积分方法时自动重置布料状态。

# 6. 操作说明

- 鼠标右键拖动：旋转相机；
- 控制面板按钮：切换三种积分方法；
- `Pause Simulation`：暂停模拟；
- `Resume Simulation`：恢复模拟；
- `Reset Cloth`：恢复初始布料状态。
